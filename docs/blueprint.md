# **App Name**: Rizer Store

## Core Features:

- Secure User Authentication: Enables users to register and log in securely, featuring unique usernames and emails, password hashing, and persistent login sessions.
- Dynamic Product Catalog: Administrators can add, update, and remove digital gaming products (game accounts, gift cards) including details like name, price, image uploads, and custom tags, which are displayed on the public store pages.
- User Wallet and Fund Requests: Each user maintains a wallet balance. Users can submit fund requests by choosing a payment method, specifying an amount, and uploading a payment proof image. These requests are routed to the Admin Panel for review.
- Streamlined Purchase System: Facilitates item purchases using wallet funds, checks for sufficient balance, collects user contact details, deducts funds, records the transaction, and generates an automated WhatsApp redirect for product delivery.
- Comprehensive Admin Panel: A secure, separate interface for administrators to manage users (view balances/history), approve/reject fund requests, add/edit products, manage announcements, and oversee all purchase orders.
- WhatsApp Integration Gateway: Incorporates a floating WhatsApp support button visible on all pages and triggers an automated WhatsApp chat redirection with pre-filled message upon successful product purchases.
- Real-time Data Sync: Ensures all data (products, wallet balances, user activity, announcements, fund requests) updates live from the backend database across both user and admin interfaces.

## Style Guidelines:

- Color scheme: Dark theme with neon accents, chosen to evoke a futuristic gaming vibe. Primary color (interactable elements, highlights): vibrant blue-violet (#9D1CFF), offering a strong contrast against dark backgrounds and a glowing appearance. Background color: deep, desaturated purple (#1C181F), providing a fitting dark canvas for neon elements. Accent color (secondary highlights, subtle emphasis): Electric Blue (#3F3FDF), complementing the primary while adding dynamic visual interest.
- Headlines: 'Space Grotesk' (sans-serif) for a modern, tech-inspired, and impactful look suitable for titles and prominent text. Body text: 'Inter' (sans-serif) for high readability and a clean, objective aesthetic, ensuring comfort in longer descriptions and content blocks.
- Utilize sleek, modern, and geometrically styled icons with a slight neon glow effect to align with the gaming aesthetic. Emphasize crisp lines and minimal details for clarity within the dark theme.
- Implement a fully responsive design ensuring an optimal viewing experience on both desktop and mobile devices. Adopt a clean, spacious layout for dashboards and product displays, using 'modern product cards' with prominent imagery and clear calls to action, all framed within a dark, gaming-inspired aesthetic.
- Incorporate smooth, subtle transition animations for page navigation, element interactions, and content loading. Design interactive elements like buttons and product cards with 'glowing effects' and fluid hover states to enhance the dynamic, neon gaming atmosphere.
